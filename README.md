## ‚ú® Core Features

-   **üñºÔ∏è Full Resolution Spectrum:** Download thumbnails in every available format, including:
    -   **4K / Max Resolution** (`maxresdefault`, 1920x1080)
    -   **Full HD** (`hq720`, 1280x720) - *The recommended standard!*
    -   **Standard Definition** (`sddefault`, 640x480)
    -   **High & Medium Quality** (`hqdefault` / `mqdefault`)
-   **üîó Multi-Format URL Support:** Paste virtually any YouTube link format and it will be recognized:
    -   Standard Watch URL (`youtube.com/watch?v=...`)
    -   Shortened URL (`youtu.be/...`)
    -   Embed URL (`youtube.com/embed/...`)
    -   Shorts URL (`youtube.com/shorts/...`)
    -   Plain Video ID (`dQw4w9WgXcQ`)
-   **üß† Intelligent Fallback System:** If a requested high-resolution thumbnail doesn't exist for a particular video, the tool automatically fetches the next best available quality, ensuring you always get a valid image.
-   **üñ±Ô∏è One-Click Actions:**
    -   **Download Image:** Save the high-quality thumbnail directly to your device.
    -   **Copy URL:** Instantly copy the direct image URL to your clipboard for easy sharing or embedding.
-   **üì± Fully Responsive Design:** The interface is meticulously crafted to be comfortable and fully functional on desktops, tablets, and mobile phones.
-   **üîí Secure & Private:** Operates entirely within your browser. No data is sent to or stored on any server.

## üõ†Ô∏è Technology Stack & Architecture

This project is built with a modern, performant, and scalable tech stack, ensuring a great developer and user experience.

-   **Core Framework:** **[React 18](https://reactjs.org/)** (via Next.js) for a component-based architecture.

## üöÄ Quick Start

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (version 18.x or newer recommended)
-   `npm`, `yarn`, or `pnpm` package manager

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/youtube-thumbnail-grabber.git
    cd youtube-thumbnail-grabber
    ```
2.  **Install dependencies:**
    ```sh
    npm install
    ```
3.  **Run the development server:**
    ```sh
    npm run dev
    ```
4.  Open your browser and navigate to `http://localhost:3000`.
